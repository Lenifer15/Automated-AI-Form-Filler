# Automated AI Form Filler

## 📌 Project Overview
This project is an AI-powered system that automates the process of filling forms by extracting and mapping relevant data. It reduces manual effort, minimizes errors, and improves efficiency across repetitive data entry tasks.

---
## 🚀 Features
- Automatically fills forms using AI
- Extracts and maps data intelligently
- Reduces manual input and human errors
- Works with different form structures
- Scalable and modular design

---
## 🧠 Technologies Used
- Python
- OpenAI API
- Selenium
- Pandas
- HTML/CSS

---
## ⚙️ How It Works
1. User provides input data or document
2. System extracts relevant information using AI
3. Data is mapped to corresponding form fields
4. Automation script fills the form
5. Final output is validated and submitted
